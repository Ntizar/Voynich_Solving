El Voynich como notacion farmaceutica, no como promesa de desciframiento

La propuesta

No intentar leer primero, sino entender la forma del texto

La idea original

En lugar de asumir que el Voynich es prosa cifrada, este proyecto lo trata como algo mas cercano a un sistema de fichas, registros y campos funcionales. En esa lectura, cada linea de receta se parece menos a una frase y mas a una entrada de una base de datos medieval.

Eso permitia plantear dos preguntas separadas: si hay una estructura farmaceutica real y, solo despues, si algunos stems pueden mapearse con seguridad a ingredientes historicos.

Las dos preguntas clave

  • Estructura: la seccion de recetas se comporta como una base de datos farmaceutica?
  • Semantica: podemos demostrar que un stem concreto significa un ingrediente concreto?

La investigacion final muestra que la primera pregunta tiene una respuesta bastante fuerte. La segunda, de momento, no.

Resultado

Que se sostiene y que no

Lo que si se sostiene

  • La seccion de recetas tiene una estructura estadistica anomala frente a controles medicos latinos.
  • Hay alineacion vertical, selectividad de sufijos y variacion de esquema compatibles con notacion funcional.
  • El perfil global no encaja bien con un cifrado clasico corriente.

Lo que no hemos podido demostrar

  • Que los mappings `stem -> ingrediente` superen de forma clara a baselines triviales en test ciego.
  • Que `v7` sea una validacion independiente: estaba contaminado y varias metricas eran tautologicas.
  • Que ya podamos "leer" el manuscrito en ningun sentido fuerte.
Ultima ronda

Que se hizo estas semanas para intentar salvar la linea semantica

1. Benchmark comun

Se comparo el mismo evaluador sobre tres representaciones de entrada: STA stems, STA tokens y EVA tokens.

2. Corpus bajo lupa

Se midio el solapamiento entre recetas, la concentracion de ingredientes y el soporte real que train da a las recetas de test.

3. Fuentes externas

Se buscaron nuevas recetas en farmacopeas abiertas y se monto un corpus aumentado paralelo con testigos de Amsterdam.

La expansion externa y el cambio de representacion no fueron inutiles: sirvieron para poner a prueba la hipotesis con mas dureza. Pero no han dado la clase de victoria ciega que haria falta para hablar de desciframiento.

Estado y proximo paso

La situacion natural del proyecto ahora mismo

Si se cierra aqui

Se puede cerrar bien y con valor real. El proyecto dejaria una tesis estrecha pero fuerte: la seccion de recetas del Voynich parece una notacion farmaceutica estructurada y no una prosa cifrada cualquiera.

Esa conclusion ya justifica publicar, resumir o presentar el trabajo sin inflar la parte semantica.

Si algun dia se reabre

Solo deberia reabrirse con una regla simple: no volver a hablar de lectura hasta que una evaluacion ciega gane de forma clara a la mejor baseline y mantenga esa ventaja al repetir el experimento.

Si eso no pasa, la interpretacion correcta sigue siendo estructural, no semantica.
